dilli.today is an [open-source](/license) project that curates events happening in Delhi. It curates events from multiple sources, cleans them up, then curates them further by tagging them nicely, and 
makes all event data available as calendars you can subscribe to.

It is inspired by and a (vibe-coded) fork of the [blr.today](https://blr.today) project by [Nemo](https://github.com/captn3m0). You can find the thesis, roadmap and history of the project at [blr.today/about](https://blr.today/about).
